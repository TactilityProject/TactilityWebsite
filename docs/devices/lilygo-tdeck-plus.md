# LilyGo T-Deck Plus

Note: The internal GPS is wired to the Grove port, so the Grove port cannot be used.

## Features

- SD card
- Power status
- USB Storage
- Keyboard (backlight is supported on newer hardware)
- Trackball
- GPS
